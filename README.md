# Security-Checklist-Pro 🛡️

> **Compliance (CIS, NIST, ISO) is a nightmare of spreadsheets. We made it a dream of interactivity.**

**Security-Checklist-Pro** is a premium, gamified dashboard designed for security engineers, auditors, and IT professionals to track system hardening and compliance progress without the "Excel fatigue."

![Preview Placeholder](https://via.placeholder.com/1200x600/05070a/00ffc3?text=Security-Checklist-Pro+Dashboard)

## ✨ Key Features

- **🏆 Gamified Progress:** Watch your "Security Score" climb as you harden your infrastructure.
- **📊 Multi-Framework Support:** Built-in checklists for **CIS Benchmarks**, **NIST CSF 2.0**, and **ISO 27001**.
- **💾 Auto-Save:** Progress is saved automatically to `LocalStorage`. No database, no login, total privacy.
- **📄 Pro Reporting:** Export your status as a clean **Markdown** document or a professional **PDF Report**.
- **💎 Premium UI:** Glassmorphic design, smooth CSS transitions, and high-readability typography.
- **🚀 Zero Friction:** Static site—deploy to GitHub Pages in seconds.

## 🛠️ Tech Stack

- **Core:** Vanilla JavaScript (ES6+)
- **Styling:** CSS3 (Modern Flex/Grid, Glassmorphism, Print Styles)
- **Data:** JSON-driven framework definitions
- **Icons:** SVG-based lightweight iconography

## 🚀 Quick Start

1. **Clone the repo:**
   ```bash
   git clone https://github.com/your-username/Security-Checklist-Pro.git
   ```
2. **Open `index.html`** in any modern browser.
3. **Start Hardening!**

## 📖 Deployment

This repository is optimized for **GitHub Pages**. 
To deploy:
1. Push this code to a GitHub repository.
2. Go to **Settings > Pages**.
3. Select `Main` branch and `/root`.
4. Your dashboard will be live!
